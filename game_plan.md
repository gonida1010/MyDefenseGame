# ⚔️ [기획서] 귀멸의 칼날 랜덤 디펜스 (Solo)

## 1. 프로젝트 개요

- **프로젝트명:** 귀멸의 칼날 랜덤 디펜스 (Solo)
- **장르:** 랜덤 디펜스 / 전략 시뮬레이션
- **플랫폼:** Web (PC 브라우저)
- **배경:** 도깨비들이 활동하는 어두운 **밤**
- **개발 도구:** [Phaser.js](https://phaser.io/)

## 2. 게임 메카닉 (Game Mechanics)

### 🔹 소환 및 성장 시스템

1. **랜덤 소환:** 플레이어는 재화를 지불하여 티어별 유닛을 랜덤하게 생성.
2. **합성 (Merge):**
   - 동일한 유닛을 합성하여 **다음 티어** 유닛으로 승급. => 수정: **전체** 티어 조합법이 있음.
3. **조합 (Recipe):**
   - 고위 티어 유닛은 단순 합성이 아닌, 특정 유닛들의 **조합식(Recipe)** 을 통해 획득.
   - 조합 난이도를 고려하여 전체 라운드 구성은 여유 있게 설정.
     **히든조합법:**
   - 조합법에는 안 나오는 히든 조합법이 숨겨져 있음.

### 🔹 전투 시스템

1. **승리 조건:** 최종 보스 **'키부츠지 무잔'** 처치.
   - 라운드 무한 모드 진행 예정
2. **패배 조건:** 제한 시간 내에 적을 처치하지 못해 필드 허용량을 초과할 경우 (타임아웃).
3. **맵 구조:**
   - **원형 맵:** 적이 특정 경로를 이동하는 것이 아니라, 맵 안에 생성되어 쌓이는 구조.
   - **타임아웃 방식:** 입구→출구 통과 방식이 아닌, 섬멸전(Time Attack) 형태.
4. **속성:** 캐릭터별 고유 속성(호흡) 보유, 적과의 상성 관계 존재.

### 🔹 편의 기능 (솔로 모드 전용)

- **일시 정지 (Time Stop):**
  - 전략적 판단을 위해 시간을 멈출 수 있음.
  - 기능 활성화 시 화면이 **어둡게(Dimmed)** 변하며 정지 상태임을 시각적으로 알림.
- **멀티플레이:** 추후 업데이트 예정.

## 3. 레벨 및 몬스터 디자인

### 👹 적(Enemy) 구성

- **일반 적:** 하현(下弦) 및 하급 도깨비
- **보스:** 상현(上弦) 도깨비
- **최종 보스:** 키부츠지 무잔

### ⏳ 라운드 구성

- **보스 출현 주기:** **10라운드**마다 강력한 보스(상현 등) 등장.
- 전체적으로 조합 시간을 벌 수 있도록 긴 호흡의 라운드 구성.

## 4. 기술 스택 (Tech Stack)

### 🛠️ Phaser.js 프레임워크

- **선정 이유:** 전 세계적으로 가장 널리 사용되는 JavaScript 2D 게임 엔진.
- **주요 장점:**
  - 스프라이트(Sprite), 충돌 처리(Physics), 카메라(Camera) 등 필수 기능 내장.
  - 웹 기반으로 접근성이 좋으며 빠른 프로토타이핑 가능.
