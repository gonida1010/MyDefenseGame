# 귀멸의 칼날 랜덤 디펜스 (Demon Slayer Random Defense)
## 게임이 매우 어려울 수 있습니다. 조합법을 먼저 보시는거 추천해요!

인기 애니메이션 **'귀멸의 칼날'** IP를 활용하여 제작한 웹 기반 랜덤 디펜스 게임입니다.
**Phaser.js** 게임 엔진을 사용하여 개발되었으며, **Firebase**를 연동하여 실시간 랭킹 시스템을 지원합니다.

## 🎮 게임 소개

플레이어는 금화를 사용하여 대원을 랜덤하게 소환하고, 같은 등급의 대원을 조합하여 더 강력한 상위 등급의 대원(지주, 혈귀술 등)으로 성장시켜야 합니다. 몰려오는 혈귀들을 막아내고 최종 보스 **'키부츠지무잔'**을 토벌하는 것이 목표입니다.

## ✨ 주요 기능

* **랜덤 소환 시스템:** 골드를 소비하여 확률에 따라 1~3티어 유닛을 소환합니다.
* **조합(Recipe) 시스템:** 특정 유닛 2기를 조합하여 상위 티어 유닛을 제작합니다. (히든 조합 존재!)
* **화려한 연출:**
    * 보스 등장 시 전용 컷신(Cutscene) 및 경고 효과 재생
    * 각 캐릭터별 고유의 호흡/혈귀술 이펙트 및 투사체 구현
* **보스 레이드:** 10라운드마다 강력한 십이귀월 보스가 등장합니다.
* **편의 기능:** 게임 속도 조절 (1배속 / 2배속), 유닛 판매, 조합법 확인 팝업.
* **랭킹 시스템:** Firebase Firestore를 연동하여 플레이어의 최고 라운드 기록을 저장하고 공유합니다.

## 🛠 기술 스택 (Tech Stack)

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Game Engine:** Phaser 3
* **Backend / Database:** Firebase Firestore (Leaderboard)

# 조합법 가이드 (Recipe Book)

본 게임의 조합 시스템은 **100% 확률**로 성공하며 실패가 없습니다.
동일한 재료 조합이 겹치지 않도록 설계되었습니다.

---

## 1단계: 기초 조합 (Basic)
**T1(기초 재료) + T1(기초 재료) ➔ T2(일반 대원)**

가장 기초가 되는 3가지 재료(일반 대원, 까마귀, 은)를 통해 6명의 캐릭터를 생성합니다.

| 재료 A | 재료 B | 결과 (T2) | 비고 |
|:---:|:---:|:---|:---|
| 일반 대원 | 까마귀 | **탄지로** | |
| 까마귀 | 까마귀 | **젠이츠** | |
| 일반 대원 | 일반 대원 | **이노스케** | |
| 일반 대원 | 은(Kakushi) | **겐야** | |
| 까마귀 | 은(Kakushi) | **카나오** | |
| 은(Kakushi) | 은(Kakushi) | **아오이** | |

---

## 2단계: 성장 조합 (Growth)
**T2(일반 대원) + T1(기초 재료) ➔ T3(숙련자/조력자)**

핵심 인물들이 기초 재료를 만나 성장하거나, 그들의 조력자를 만납니다.

| 재료 A | 재료 B | 결과 (T3) | 설명 |
|:---:|:---:|:---|:---|
| 탄지로 | 일반 대원 | **우로코다키** | 탄지로의 스승 |
| 탄지로 | 은 | **네즈코 상자** | 상자 안에 든 네즈코를 업고 다님 |
| 젠이츠 | 일반 대원 | **쿠와지마** | 젠이츠의 스승(할아버지) |
| 이노스케 | 은 | **하가네즈카** | 이노스케의 검을 수리(분노) |
| 겐야 | 까마귀 | **무라타** | 끈질긴 생존력의 대원 |
| 카나오 | 일반 대원 | **사비토** | 수련을 도와준 영혼 |
| 아오이 | 겐야 | **유시로** | 오니의 힘을 가진 조력자 |

---

## 3단계: 지주 조합 (Hashira)
**T3 + T2 ➔ T4(지주/상위 등급)**

귀살대의 최고 전력인 지주(Hashira)와 특수 캐릭터 조합입니다.

| 재료 A (T3) | 재료 B (T2) | 결과 (T4) | 설명 |
|:---:|:---:|:---|:---|
| 우로코다키 | 사비토 | **기유 (수주)** | 스승과 친구의 유대 |
| 네즈코 상자 | 탄지로 | **렌고쿠 (염주)** | 무한열차의 수호자 |
| 무라타 | 젠이츠 | **텐겐 (음주)** | 잠입 수사와 청각의 조화 |
| 네즈코 상자 | 아오이 | **시노부 (충주)** | 오니를 연구하는 나비저택 |
| 사비토 | 카나오 | **카나에 (화주)** | 영혼으로 이어진 자매와 인연 |
| 사비토 | 겐야 | **무이치로 (하주)** | 천재적인 재능과 노력 |
| 하가네즈카 | 탄지로 | **미츠리 (연주)** | 대장장이 마을의 인연 |
| 무라타 | 이노스케 | **오바나이 (사주)** | 하급 대원 훈련과 뱀 같은 움직임 |
| 쿠와지마 | 겐야 | **사네미 (풍주)** | 엄격한 스승과 독한 마음 |
| 우로코다키 | 쿠와지마 | **교메이 (암주)** | 연륜 있는 스승들의 합작 |
| 유시로 | 아오이 | **타마요** | 충심과 의학 지식의 결합 |

---

## 4단계: 각성 조합 (Awakening)
**T4 + (T3 or T2) ➔ T5(각성)**

지주급 캐릭터가 한계를 뛰어넘어 각성합니다.

| 재료 A (T4) | 재료 B | 결과 (T5) | 설명 |
|:---:|:---:|:---|:---|
| 렌고쿠 | 탄지로 | **탄지로 (태양)** | 히노카미 카구라 계승 |
| 텐겐 | 쿠와지마 | **젠이츠 (신속)** | 벽력일섬 신속 |
| 네즈코 상자 | 미츠리 | **네즈코 (각성)** | 폭혈 |
| 기유 | 탄지로 | **기유 (반점)** | 반점 발현 |
| 무이치로 | 하가네즈카 | **무이치로 (반점)** | 기억을 되찾고 반점 발현 |
| 타마요 | 유시로 | **타마요 & 유시로** | 무잔을 옥죄는 콤비 |
| 시노부 | 이노스케 | **이노스케 (각성)** | 상현 2를 쓰러뜨린 인연 |

## 5단계: 신화 조합 (Mythic)
**T5 + T5 (일부 예외) ➔ T6(최종)**

세계관 최강의 힘을 가진 신화 등급 캐릭터입니다.

| 재료 A | 재료 B | 결과 (T6) | 설명 |
|:---:|:---:|:---|:---|
| 탄지로(태양) | 기유(반점) | **탄지로 (13형)** | 무잔을 베는 13번째 형 |
| 탄지로(태양) | 네즈코(각성) | **요리이치** | 전설의 시작 |
| 교메이(암주) | 무이치로(반점) | **교메이 (반점)** | [예외] T4 + T5 조합 |
| 이노스케(각성) | 젠이츠(신속) | **이노스케 (산의 왕)** | 친구들이 도달한 정점 |
