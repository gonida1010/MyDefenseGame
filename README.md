# 귀멸의 칼날 랜덤 디펜스 (Demon Slayer Random Defense)

> **"마음을 불태워라. 한계는 없다."**
>
> 인기 애니메이션 **'귀멸의 칼날'** IP를 활용한 팬메이드 웹 랜덤 디펜스 게임입니다. **Phaser.js**로 구현된 타격감 있는 전투와 **Socket.IO**를 이용한 **실시간 멀티플레이**를 지원합니다.


## 🎮 게임 모드 (Game Modes)

본 게임은 플레이어의 숙련도에 따라 3가지 모드를 제공합니다.

### 📘 스토리 모드 (Normal)
* **목표:** 90라운드 최종 보스 **'키부츠지 무잔'** 토벌.
* **특징:**
    * 초심자를 위한 표준 난이도.
    * 보스를 처치하면 감동적인 **엔딩 크레딧**과 함께 게임이 클리어됩니다.
    * 별도의 클리어 랭킹이 기록됩니다.

### 📕 지옥 모드 (Hard)
* **목표:** **무한 라운드** 생존 (랭커 도전).
* **특징:**
    * **초반부터 강력함:** 적 체력이 1.5배 높게 시작하며, 10라운드부터 급격히 강해집니다.
    * **지옥의 난이도:** 90라운드 이후부터는 체력이 기하급수적(복리)으로 증가하여, 한순간의 방심이 패배로 이어집니다.
    * 끝이 없습니다. 당신의 한계를 시험하세요.
 
### 💜 협동 모드 (Multiplayer) [NEW!]
* **목표:** 동료와 함께 협력하여 적을 막아내세요.
* **특징:**
    * **Socket.IO** 기반 실시간 2인 협동 플레이.
    * **[동기화 시스템]:** 적 생성, 이동, 체력, 사망 여부가 실시간으로 공유됩니다.
    * **[공동 전투]:** 내가 설치한 유닛을 동료의 화면에서도 볼 수 있으며, 함께 공격합니다.

## ✨ 주요 기능

* **전략적 랜덤 소환:** 골드를 모아 일반~희귀 등급 대원을 소환합니다.
* **확정 조합 시스템 (Recipe):** 확률 싸움은 그만! **100% 성공**하는 조합법으로 상위 유닛을 제작하세요.
* **쾌적한 조작감:**
    * **스페이스바(Space):** 유닛 즉시 판매 단축키 지원.
    * **배속 모드:** 1배속 / 2배속 토글 기능으로 속도감 있는 플레이.
    * **가이드 팝업:** 인게임에서 언제든 조합법 확인 가능.
* **화려한 연출:**
    * 전용 타격 이펙트 및 투사체 애니메이션.
    * 십이귀월 보스 등장 시 전용 **컷신(Cutscene)** 및 경고 연출.
    * 메뉴 화면의 부드러운 배경 전환 효과.

## 🛠 기술 스택 (Tech Stack)

* **Frontend:** HTML5, CSS3, JavaScript (ES6 Modules)
* **Game Engine:** Phaser 3
* **Backend:** Firebase Firestore (Realtime Leaderboard)
* **Deployment:** GitHub Pages / Netlify
